## Comment modifier les vidéos

Pour changer une vidéo YouTube :
1. Allez sur YouTube et trouvez votre vidéo
2. Copiez l'ID de la vidéo (la partie après `watch?v=` dans l'URL)
3. Remplacez l'ID dans le code ci-dessous (ex: `dQw4w9WgXcQ`)

      src="https://www.youtube.com/embed/VOTRE_VIDEO_ID"

      allowfullscreen>
    </iframe>
  </div>
